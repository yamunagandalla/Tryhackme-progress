✅ Intro to LAN

🔗 https://tryhackme.com/room/introtolan

This room introduces the fundamental concepts of Local Area Networks (LANs) and how devices communicate within a network.

It covers key networking topics such as network topologies (Bus, Star, Ring), IP addressing, subnetting, ARP (Address Resolution Protocol), and DHCP (Dynamic Host Configuration Protocol).

The room builds a strong foundation in understanding how devices identify, locate, and communicate with each other inside a local network — an essential skill for cybersecurity and ethical hacking.

The focus is on understanding real-world networking structures and how these concepts support security, traffic analysis, and network defense.
